# Dex3-1 关节电机顺序

Source: <https://support.unitree.com/home/zh/G1_developer/joint_motor_sequence>

## Dex3-1 关节电机顺序

`unitree_hg::msg::dds_::HandCmd_.motor_cmd` 与 `unitree_hg::msg::dds_::HandState_.motor_state` 包含所有的灵巧手电机的信息，其电机顺序如下：

| Hand Joint Index in IDL | Hand Joint Name |
| --- | --- |
| 0 | thumb\_0 |
| 1 | thumb\_1 |
| 2 | thumb\_2 |
| 3 | middle\_0 |
| 4 | middle\_1 |
| 5 | index\_0 |
| 6 | index\_1 |
