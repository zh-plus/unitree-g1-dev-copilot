# RobotStateClient 类

Source: <https://support.unitree.com/home/zh/G1_developer/robot_state_client_interface>

## RobotStateClient 类

RobotStateClient 是设备状态服务提供的 Client，通过 RobotClient 可以方便地通过 RPC 方式实现对 G1 内部的服务控制、获取服务状态、设备状态和系统资源使用信息等功能（部分功能接口暂未开放）。

本接口复用自B2的设备状态服务接口。您可以在SDK的以下位置找到相关资源。

```
unitree_sdk2\include\unitree\robot\b2\robot_state //C++
unitree_sdk2py\b2\robot_state #Python
unitree_ros2\example\src\include\common\ros2_robot_state_client.h //ROS2
```

### 接口错误码列表：

| 错误号 | 错误描述 | 备注 |
| --- | --- | --- |
| 5201 | 服务开关执行错误。 | 服务端返回 |
| 5202 | 服务受保护，不允许开启或关闭。 | 服务端返回 |

### 类与接口描述：

| 类名 | 构造与析构 |
| --- | --- |
| RobotStateClient | explicit RobotStateClient(); ~RobotStateClient(); |

| **函数名** | **ServiceSwitch** |
| --- | --- |
| **函数原型** | int32\_t ServiceSwitch(const std::string& name, int32\_t swit, int32\_t& status) |
| **功能概述** | 服务开关。 |
| **参数** | **name**：服务名   **swit**：开关，取值 1 开启，0 关闭。   **status**：操作执行后的服务状态（0：已开启，1：已关闭） |
| **返回值** | 调用成功返回 0，否则返回相关错误码。 |
| **备注** |  |

| --- | --- |

| **参数** | **interval**：设置上报时间间隔，单位秒；  **duration**：设置持续时间，单位秒。 |
| **返回值** | 调用成功返回 0，否则返回相关错误码。 |
