# 概念说明

| 概念 | 说明 |
| --- | --- |
| 零力矩模式 | 机器人所有电机停止主动运动，摆动时没有阻尼感。 |
| 阻尼模式 | 机器人所有电机停止主动运动，摆动时有明显阻尼感，可以进入预备模式 |
| 预备模式 | 机器人将在5秒内缓慢摆出运动模式前的准备姿势（无平衡控制） |
| 下蹲模式 | 机器人将在5秒内缓慢摆出下蹲姿势（无平衡控制） |
| 落座模式 | 机器人将在5秒内缓慢摆出落座姿势（无平衡控制） |
| 运动模式 | 可以通过遥控器控制机器人移动的模式 |
| 持续走路模式 | 机器人始终处于踏步状态 |
| 站立模式 | 该模式下，当摇杆指令为零时，机器人停止踏步，进入站立；当摇杆指令不为零，或机器人受到干扰难以维持平衡时，机器人将开始迈步。 |
| 调试模式 | 用于底层开发。在需要使用 SDK 进行开发调试时，请务必确保 G1 已经进入调试模式(阻尼或零力矩模式下，遥控器按L2+R2进入调试模式)，以停止运动控制程序发送指令，这样可以避免潜在的指令冲突问题。您可以按下 L2+A 来确认是否进入了调试模式。调试中，若出现紧急情况，请通过遥控器按 L2 + B 让设备进入阻尼状态。 |

> 机器人行走模式，暂未部署爬楼梯的功能，请勿随意爬楼梯，避免损伤机器人。

# 模式切换

![](https://oss-global-cdn.unitree.com/static/1de228d539a847dd89b0a3145570fb16_11798x7046.jpg)

注意

1、从**主运控**通过 **L2 + A** 切回**蹲姿**状态后，在不关机的情况下，如果还想切**回主运控**模式，需要通过 **L2 + B** 进入**阻尼**模式，再通过 **L2 + A** 才能切回**主运控**。

2、设备部仅支持在`零力矩`或者`阻尼`状态下，按`L2+R2`才能进入`调试模式`

3、在站立状态下，部分按键需要组合`长按2秒`才能生效。

4、**L2 + B** `阻尼按键，作为急停按键，在调试状态下依然有效, 能有效阻止机器人状态失控。`

# 按键说明

| 切换模式/功能 | 按键 |
| --- | --- |
| 调试模式 | 长按 L2 + 单击 R2 |
| 零力矩模式 | 长按 L2 + 单击 Y |
| 阻尼模式 | **①** 长按 L2 + 单击 B |
| 锁定站立 | **②** 长按 L2 + 单击 UP |
| 落座模式 | **④** 长按 L2 + 单击 LEFT |
| 躺卧站立 | **⑤** 长按 L2 + 单击 X |
| 蹲站切换 | **⑥**长按 L2 + 单击 A |

| 交互功能 | 按键 |
| --- | --- |
| 挥手 | 长按 SELECT + 单击 Y |
| 握手 | 长按 SELECT + 单击 A |
| 转身挥手 | 长按 SELECT + 单击 X |

| **③** R2 + A（走跑运控） | 按键 |
| --- | --- |
| 慢速跑 | 长按 R2 + DOWN |
| 快速跑 | 长按 R2 + UP |
| 前倾补尝 | 长按 START + 长按 UP |
| 后倾补尝 | 长按 START + 长按 DOWN |

| **⑦** R1 + X （常规运控） | 按键 |
| --- | --- |
| 持续踏步(不推荐) | 双击 START |
| 站立 | 双击 START |
| 低速模式 | 双击 L2 |
| 高速模式 | 双击 L1 |
| 偏移补偿 | 左偏：长按 R1 + 单击 →  右偏：长按 R1 + 单击 ←  前偏：长按 R1 + 单击 ↓ 后偏：长按 R1 + 单击 ↑ |

| **⑧** R1 + Y （仅三自由度腰部结构) | 按键 |
| --- | --- |
| 持续踏步(不推荐) | 双击 START |
| 站立 | 双击 START |
| 低速模式 | 双击 L2 |
| 高速模式 | 双击 L1 |
| 偏移补偿 | 左偏：长按 R1 + 单击 →  右偏：长按 R1 + 单击 ←  前偏：长按 R1 + 单击 ↓ 后偏：长按 R1 + 单击 ↑ |

> * **⑧**仅适用于腰部三自由度结构，推荐使用 R1 + Y 模式
> * 常规开机流程： 开机->**①**->**②**->**③**->演示->**④**(椅子落座)->关机
> * 躺倒开机流程：开机（确保设备胯部与地面贴平)->**①**->**⑤**->演示->**⑥**->关机
> * 蹲姿开机流程：开机(蹲姿状态)->**①**->**⑥**->演示->**⑥**->关机
> * 使用保护架开关机流程，请参考[《快速开始》](https://support.unitree.com/home/zh/G1_developer/quick_start)

注意 ！！！

1、躺姿开机和蹲姿开机仅适用于平整、坚硬地面，其他场景有摔倒风险，请勿随意尝试。
2、装有灵巧手设备，请慎用躺姿、蹲姿开机方式，有可能对灵巧手造成损坏。
