# 手臂动作服务接口

Source: <https://support.unitree.com/home/zh/G1_developer/arm_action_interface>

## 手臂动作服务接口

宇树为G1添加了许多基于上肢动作的互动功能，由手臂动作服务管理。手臂动作服务（Arm Action Client）即为调用这些上肢的互动功能提供接口。

注意

手臂动作服务接口依赖于内置运控，进入调试模式后内置运控完全退出，手臂动作服务失效。
请在确保周围安全的情况下使用手臂动作服务接口，特别是触发示教动作的播放，预留足够的展示空间。

## RPC接口调用方式

用户可以创建unitree::robot::g1::ArmAction 对象，通过成员函数发送请求，内置的手臂动作服务响应请求，执行操作。

```
#include <unitree/robot/g1/arm/g1_arm_action_client.hpp>
#include <unitree/robot/g1/arm/g1_arm_action_api.hpp>
```

## 接口介绍

| **函数名** | **Execute Action** |
| --- | --- |
| 函数原型 | `int32_t ExecuteAction(int32_t action_id)` |
| 功能概述 | 执行预设的手臂交互动作。 |
| 参数 | id：手臂动作ID。 |
| 返回值 | 调用成功返回 0，否则返回相关错误码。 |
| 备注 | 阻塞执行。手臂动作ID见前文《动作ID》。此描述适用于传入参数格式为int时。 |

| **函数名** | **Execute Action** |
| --- | --- |
| 函数原型 | `int32_t ExecuteAction(const std::string &action_name)` |
| 功能概述 | 执行预录制完毕的示教动作。 |
| 参数 | action\_name：示教动作的保存名称。 |
| 返回值 | 调用成功返回 0，否则返回相关错误码。 |
| 备注 | 非阻塞执行。用户使用App中的示教功能录制的动作名称，区分大小写。此描述适用于传入参数格式为string时。 |

| **函数名** | **Stop Custom Action** |
| --- | --- |
| 函数原型 | `int32_t StopCustomAction()` |
| 功能概述 | 停止执行示教动作。 |
| 参数 | 无 |
| 返回值 | 调用成功返回 0，否则返回相关错误码。 |
| 备注 | 停止后手臂回到初始位置。 |

| **函数名** | **Get Action List** |
| --- | --- |
| 函数原型 | `int32_t GetActionList(std::string &data)` |
| 功能概述 | 列出当前可用的动作列表。 |
| 参数 | data：动作列表（包含可使用的动作、特殊动作对FsmID的要求、示教动作的名称和时间） |
| 返回值 | 调用成功返回可调用动作列表，调用失败返回错误码。 |
| 备注 | 以服务端返回的为准。 |

## 错误码

| **错误码** | **含义** | **备注** |
| --- | --- | --- |
| 7400 | 话题被占用 | 有动作正在执行 |
| 7401 | 手臂正举起，请使用动作ID99或之前使用的相同ID来恢复手臂到初始状态。 | 适用于平局、比心等持续动作 |
| 7402 | 动作ID不存在 |  |
| 7404 | 当前FsmID不可触发此动作。 | 部分动作在走跑运控下不可触发 |

## 关闭手臂动作服务

如需通过 `/arm_sdk`话题自主开发上肢动作，必须先关闭宇树内置的手臂控制服务。

通过[设备状态服务接口](https://support.unitree.com/home/zh/G1_developer/robot_state_client_interface)可以关闭宇树内置的手臂控制服务。手臂动作服务的服务名称为：`g1_arm_example`。

示例：

```
#include <unitree/robot/b2/robot_state/robot_state_client.hpp>
#include <unitree/common/time/time_tool.hpp>

using namespace unitree::common;
using namespace unitree::robot;
using namespace unitree::robot::b2;

int main(int32_t argc, const char** argv)
{
    if (argc < 2)
    {
        std::cout << "Usage: turn_off_arm_example [NetWorkInterface(eth0)]" << std::endl;
        exit(0);
    }

    std::string networkInterface = "eth0", serviceName = "g1_arm_example";

    if (argc > 1)
    {
        networkInterface = argv[1];
    }

    std::cout << "NetWorkInterface:" << networkInterface << std::endl;
    std::cout << "Switch ServiceName:" << serviceName << std::endl;

    ChannelFactory::Instance()->Init(0, networkInterface);

    RobotStateClient rsc;
    rsc.SetTimeout(10.0f);
    rsc.Init();

    std::string clientApiVersion = rsc.GetApiVersion();
    std::string serverApiVersion = rsc.GetServerApiVersion();

    if (clientApiVersion != serverApiVersion)
    {
        std::cout << "client and server api versions are not equal." << std::endl;
    }
    int status = 0;
    while (1)
    {
        int ret = rsc.ServiceSwitch(serviceName, 0, status);
        if (ret == 0 && status != 0)
        {
            std::cout << "Call ServiceSwitch[" << serviceName << ",0] ret:" << ret << ", status:" << status << std::endl;
            break;
        }
    }
    ChannelFactory::Instance()->Release();
    return 0;
}
```
