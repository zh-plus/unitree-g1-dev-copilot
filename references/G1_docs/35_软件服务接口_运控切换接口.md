# MotionSwitcherClient 类

Source: <https://support.unitree.com/home/zh/G1_developer/motion_witcher_service_interface>

## MotionSwitcherClient 类

MotionSwitcherClient 是运控切换服务提供的 Client，通过 MotionSwitcherClient 可以方便地通过 RPC 方式实现释放和恢复 G1 运控模式，释放运控模式后进入用户开发的调试模式。

通过 `#include <unitree/robot/b2/motion_switcher/motion_switcher_client.hpp>`  使用该 MotionSwitcherClient 类。
